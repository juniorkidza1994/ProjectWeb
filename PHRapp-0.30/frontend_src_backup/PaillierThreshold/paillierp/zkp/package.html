Provides the structures and methods to produce non-interactive Zero Knowledge Proofs of encryption, decryption, and multiplication in the generalized Paillier Encryption scheme. These Zero Knowledge Proofs use the Fiat-Shamir heuristic and a hash function to make the protocol non-interactive. The Zero Knowledge Proofs provided are
  • Encryption of a message m and random number r.
  • Multiplication of the message m in an encryption E(m, r) by a constant α to produce E(αm, r').
  • Partial Decryption of a ciphertext c to the partial decryption ci. (Threshold scheme, only.)

Primary Papers

  • A Generalization of Paillier's Public-Key System with Applications to Electronic Voting by Damgård, Jurik, and Nielsen, 2000. This was used for the partial decryption proof.
  • Multiparty Computation from Threshold Homomorphic Encryption by Cramer, Damgård, and Nielsen, 2000. This was used for the multiplication and encryption proofs.
